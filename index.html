INSTRUCTIONS

Memory function

Press < SHIFT > followed by < M > to load current result displayed into the memory.

Press < M > to recall number stored in memory for calculation.

Press < SHIFT > followed by < CE > to clear the memory (for the pendantic).

Exponential function

To enter a number such as eg. 1.23 x 10^4 press  < 1 >  < . >  < 2 >  < 3 >  < Exp >  < 4 > .

Answer function

Press < Ans > to use the answer from previous computation in the current one.

Percentage function

Enter an expression followed by < SHIFT > and then  < % > to use percentages in the following manner:

  • Calculate three quarters as a percentage | Result: 75 (%)

  • 3 < ÷ > 4 < SHIFT >  < % > 

  • Calculate 17.5% of $250.00 | Result: ($) 43.75

  • 250  < x > 17.5 < SHIFT >  < % > 

  • Calculate 12% mark-up on $250.00 | Result: ($) 280

  •  250 < + > 12 < SHIFT >  < % > 

  • Give 10% discount on goods costing $150 | Result: ($) 135

  • 150  < - > 10 < SHIFT >  < % >
