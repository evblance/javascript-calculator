INSTRUCTION MANUAL

Powering On

Press < AC > to turn the calculator on. It will turn off automatically after 60 seconds of inactivity.

Memory function

Press < SHIFT > < M > to load current result displayed into the memory.

Press < M > to recall number stored in memory for calculation.

Press < SHIFT > < CE > to clear the memory.

Scientific exponent function

To enter a number such as eg. 1.23 x 10^4 press  < 1 >  < . >  < 2 >  < 3 >  < Exp >  < 4 > .

Previous Answer function

Press < Ans > to use the answer from previous computation in a new computation. Similarly, pressing an operator button after a calculation will preload the previous answer into the working expression, followed by the operator.

Percentage function

Enter an expression followed by < % > to evaluate precentages as follows:

  • Calculate three quarters as a percentage | Result: 75 (%)

  • 3 < ÷ > 4 < % > 

  • Calculate 17.5% of $250.00 | Result: ($) 43.75

  • 250  < x > 17.5 < % > 

  • Calculate 12% mark-up on $250.00 | Result: ($) 280

  •  250 < + > 12 < % > 

  • Give 10% discount on goods costing $150 | Result: ($) 135

  • 150  < - > 10 < % >
